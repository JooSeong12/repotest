linux 명령어
ls >> list 보는 명령어      la -al >> 숨겨진 파일까지 보는 것
clear >> 화면 지우기
pwd(print working directory) >> 현재 위치보여주는것
whoami >> 현재 로그인 한 사용자의 정보
cd >> 폴더 이동할 때 사용
ex) cd bin >> bin 폴더로 이동 cd / or cd .. >> 상위폴더로 이동   cd ~ >> 사용자의 홈디렉토리로 이동

mkdir(make directory) > 폴더 생성
touch >> 파일 생성  touch a.txt >> 이름과 확장자명으로 생성
rm >> 파일 삭제   rm a.txt             rm -r user >> user폴더 삭제
mv c.txt user >> c.txt를 user 폴더로 옮김
